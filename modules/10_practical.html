Application 1 — Data Cleanup Macro

Scenario: Raw data with inconsistent formatting needs cleaning
Sub CleanData() Application.ScreenUpdating = False On Error GoTo Done Dim ws As Worksheet Set ws = ActiveSheet Dim lastRow As Long lastRow = ws.Cells(ws.Rows.Count, 1).End(xlUp).Row Dim i As Long For i = 2 To lastRow ' Remove extra spaces ws.Cells(i, 1).Value = Trim(ws.Cells(i, 1).Value) ' Standardize names to Proper Case ws.Cells(i, 2).Value = StrConv(ws.Cells(i, 2).Value, vbProperCase) ' Ensure phone number format (remove non-numeric) Dim phone As String phone = ws.Cells(i, 3).Value phone = Replace(Replace(Replace(phone, "-",""), " ",""), "(","") ws.Cells(i, 3).Value = Replace(phone, ")", "") ' Flag blank critical cells in red If ws.Cells(i, 1).Value = "" Then ws.Rows(i).Interior.Color = RGB(255, 200, 200) End If Next i ws.UsedRange.EntireColumn.AutoFit MsgBox "Cleanup complete! Checked " & (lastRow-1) & " rows.", vbInformation Done: Application.ScreenUpdating = True End Sub

Application 2 — Monthly Report Generator

Scenario: Auto-build a formatted summary sheet from raw data
Sub GenerateMonthlyReport() Application.ScreenUpdating = False ' Delete old report sheet if it exists Application.DisplayAlerts = False On Error Resume Next Sheets("Monthly Report").Delete On Error GoTo 0 Application.DisplayAlerts = True ' Create new report sheet Dim rpt As Worksheet Sheets.Add After:=Sheets(Sheets.Count) ActiveSheet.Name = "Monthly Report" Set rpt = Sheets("Monthly Report") ' Header With rpt.Range("A1:E1") .Merge .Value = "Monthly Sales Report — " & Format(Date, "MMMM YYYY") .Font.Bold = True .Font.Size = 16 .HorizontalAlignment = xlCenter .Interior.Color = RGB(21, 101, 192) .Font.Color = RGB(255, 255, 255) End With ' Column headers Dim headers() As String headers = Split("Product,Jan,Feb,Mar,Total", ",") Dim j As Long For j = 0 To 4 With rpt.Cells(2, j+1) .Value = headers(j) .Font.Bold = True .Interior.Color = RGB(240, 240, 240) End With Next j ' Copy summary data from Data sheet Dim src As Worksheet Set src = Sheets("Data") Dim lastRow As Long lastRow = src.Cells(src.Rows.Count, 1).End(xlUp).Row src.Range("A2:E" & lastRow).Copy rpt.Range("A3") rpt.UsedRange.EntireColumn.AutoFit MsgBox "Report created!", vbInformation Application.ScreenUpdating = True End Sub

Application 3 — Copy Filtered Data to New Sheet

Scenario: Extract rows matching a criterion into a separate sheet
Sub ExtractByDepartment() Dim dept As String dept = InputBox("Enter department to extract:", "Filter") If dept = "" Then Exit Sub Application.ScreenUpdating = False Dim src As Worksheet, dst As Worksheet Set src = Sheets("Data") ' Create destination sheet Application.DisplayAlerts = False On Error Resume Next: Sheets(dept).Delete: On Error GoTo 0 Application.DisplayAlerts = True Sheets.Add After:=Sheets(Sheets.Count) ActiveSheet.Name = dept Set dst = Sheets(dept) ' Copy header row src.Rows(1).Copy dst.Rows(1) ' Find and copy matching rows Dim srcLastRow As Long srcLastRow = src.Cells(src.Rows.Count, 1).End(xlUp).Row Dim dstRow As Long: dstRow = 2 Dim i As Long For i = 2 To srcLastRow If UCase(src.Cells(i, 3).Value) = UCase(dept) Then src.Rows(i).Copy dst.Rows(dstRow) dstRow = dstRow + 1 End If Next i dst.UsedRange.EntireColumn.AutoFit MsgBox (dstRow-2) & " rows extracted for " & dept, vbInformation Application.ScreenUpdating = True End Sub

Application 4 — Export Sheet to PDF

Scenario: Auto-generate and save a PDF report
Sub ExportToPDF() On Error GoTo ErrHandler Dim ws As Worksheet Set ws = Sheets("Monthly Report") ' Build the file path with date stamp Dim savePath As String savePath = ThisWorkbook.Path & "\" & _ "Report_" & Format(Date, "yyyymmdd") & ".pdf" ' Page setup before export With ws.PageSetup .Orientation = xlLandscape .FitToPagesWide = 1 .FitToPagesTall = False .LeftMargin = Application.InchesToPoints(0.5) .RightMargin = Application.InchesToPoints(0.5) .TopMargin = Application.InchesToPoints(0.75) .PrintTitleRows = "$1:$1" ' Repeat header on every page End With ' Export to PDF ws.ExportAsFixedFormat _ Type:=xlTypePDF, _ Filename:=savePath, _ Quality:=xlQualityStandard, _ OpenAfterPublish:=True ' Auto-open the PDF MsgBox "PDF saved: " & savePath, vbInformation Exit Sub ErrHandler: MsgBox "Failed to export PDF: " & Err.Description, vbCritical End Sub

Application 5 — Performance Optimization

Scenario: Processing 10,000+ rows — make it fast!

❌ Slow Approach (cell by cell)

' Reading/writing each cell = SLOW For i = 1 To 10000 Cells(i, 3).Value = _ Cells(i, 1).Value * _ Cells(i, 2).Value Next i ' May take 30+ seconds!

✅ Fast Approach (arrays)

' Read all to array, process, ' write back once = FAST Dim a As Variant Dim b As Variant Dim c() As Variant a = Range("A1:A10000").Value b = Range("B1:B10000").Value ReDim c(1 To 10000, 1 To 1) For i = 1 To 10000 c(i,1) = a(i,1) * b(i,1) Next i Range("C1:C10000").Value = c ' Runs in under 1 second!
🚀 Performance Checklist ☑ Application.ScreenUpdating = False
☑ Application.Calculation = xlCalculationManual
☑ Application.EnableEvents = False
☑ Read to array, process, write back
☑ Use .Value2 instead of .Value (slightly faster)
☑ Avoid .Select and .Activate — reference directly!

Application 6 — Worksheet Event Macros

Scenario: Automatic actions triggered by Excel events
📌 Where to write event codeDouble-click the Sheet name (or ThisWorkbook) in Project Explorer — not a regular Module!
' === In Sheet code (e.g. Sheet1) === ' Run when a cell is changed by user Private Sub Worksheet_Change(ByVal Target As Range) ' Only respond to column A changes If Target.Column <> 1 Then Exit Sub ' Only one cell at a time If Target.Cells.Count > 1 Then Exit Sub Application.EnableEvents = False ' Prevent recursion! Target.Value = UCase(Target.Value) ' Auto-uppercase col A Application.EnableEvents = True End Sub ' Run when sheet is selected (activated) Private Sub Worksheet_Activate() Range("A1").Select StatusBar = "Welcome to Data Entry Sheet" End Sub ' === In ThisWorkbook code === ' Run when workbook opens Private Sub Workbook_Open() Sheets("Dashboard").Activate MsgBox "Welcome! Last opened: " & Format(Now(), "yyyy/mm/dd HH:MM") End Sub ' Run before workbook closes Private Sub Workbook_BeforeClose(Cancel As Boolean) If MsgBox("Save before closing?", vbYesNo) = vbYes Then ThisWorkbook.Save End If End Sub

Application 7 — Send Email via Outlook

Scenario: Auto-send a report email from Excel VBA
' Requires: Tools → References → Microsoft Outlook Object Library Sub SendReportEmail() Dim outlookApp As Object Dim mail As Object Set outlookApp = CreateObject("Outlook.Application") Set mail = outlookApp.CreateItem(0) ' 0 = MailItem With mail .To = "manager@yamaha-motor.com" .CC = "team@yamaha-motor.com" .Subject = "Monthly Sales Report - " & Format(Date, "MMMM YYYY") .Body = "Hi Team," & Chr(13) & Chr(13) & _ "Please find the monthly report attached." & Chr(13) & _ "Generated on: " & Format(Now(), "yyyy/mm/dd HH:MM") ' Attach the current workbook .Attachments.Add ThisWorkbook.FullName ' .Send to send immediately, .Display to review first .Display ' Opens Outlook compose window for review End With Set mail = Nothing Set outlookApp = Nothing End Sub

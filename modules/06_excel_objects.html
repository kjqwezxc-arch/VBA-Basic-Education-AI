Excel Object Model

Master Workbooks, Worksheets, Ranges, and Cells — the core of Excel VBA

The Excel Object Hierarchy

Everything in Excel VBA is an object. Objects are organized in a parent-child hierarchy:

📱 Application (Excel itself)
📁 Workbooks Collection
📒 Workbook (one file)
📄 Worksheets → Worksheet
📊 Charts → Chart
📋 Names → Name
🔗 How you navigate the hierarchy Application.Workbooks("Book1.xlsm").Worksheets("Sheet1").Range("A1").Value = "Hello"
Usually you can shorten this — VBA assumes the active workbook/sheet.

Workbooks — Managing Excel Files

' Reference workbooks Dim wb As Workbook ThisWorkbook ' The file containing this VBA code ActiveWorkbook ' Currently active file Workbooks("Report.xlsx") ' By name Workbooks(1) ' By index (first open workbook) ' Open a workbook Set wb = Workbooks.Open("C:\Data\Report.xlsx") ' Create a new workbook Set wb = Workbooks.Add ' Save workbook wb.Save wb.SaveAs "C:\Data\NewReport.xlsx" ' Close workbook wb.Close SaveChanges:=True ' Check if a workbook is open Function IsWorkbookOpen(ByVal wbName As String) As Boolean On Error Resume Next IsWorkbookOpen = (Workbooks(wbName).Name = wbName) On Error GoTo 0 End Function

Worksheets — Working with Sheets

' Reference worksheets Dim ws As Worksheet ActiveSheet ' Currently visible sheet Sheets("Sales") ' By tab name Worksheets(1) ' By index (leftmost = 1) Set ws = Sheets("Data") ' Add a new sheet Sheets.Add After:=Sheets(Sheets.Count) ActiveSheet.Name = "NewSheet" ' Loop through all sheets For Each ws In ThisWorkbook.Worksheets ws.Tab.Color = RGB(255, 143, 0) Next ws ' Check if sheet exists Function SheetExists(ByVal shName As String) As Boolean On Error Resume Next SheetExists = (Sheets(shName).Name = shName) On Error GoTo 0 End Function ' Common sheet properties ws.Visible = xlSheetHidden ' Hide sheet ws.Protect Password:="1234" ' Protect sheet

Range & Cells — Accessing Cell Data

' Range — by address Range("A1") ' Single cell Range("A1:C5") ' Block of cells Range("A:A") ' Entire column A Range("1:1") ' Entire row 1 ' Cells — by row, column number Cells(1, 1) ' A1 (row 1, col 1) Cells(3, 4) ' D3 (row 3, col 4) ' Combining Range and Cells Range(Cells(1,1), Cells(5,3)) ' A1:C5 ' Read and write values Range("A1").Value = "Hello" Dim v As String v = Range("A1").Value
' Navigate relative to a cell Range("B2").Offset(1, 0) ' B3 (1 row down) Range("B2").Offset(0, 1) ' C2 (1 col right) Range("B2").Offset(-1, 0) ' B1 (1 row up) ' Find last row and column Dim lastRow As Long lastRow = Cells(Rows.Count, 1).End(xlUp).Row Dim lastCol As Long lastCol = Cells(1, Columns.Count).End(xlToLeft).Column ' Resize a range Range("A1").Resize(5, 3) ' 5 rows, 3 cols ' EntireRow and EntireColumn Range("A5").EntireRow.Delete Range("C1").EntireColumn.Hidden = True

Cell Properties — Formatting

With Range("A1:D1") ' Value and format .Value = "Monthly Report" .NumberFormat = "#,##0.00" ' Number with commas .NumberFormat = "yyyy/mm/dd" ' Date format ' Font .Font.Bold = True .Font.Size = 14 .Font.Name = "Calibri" .Font.Color = RGB(255, 255, 255) ' Background .Interior.Color = RGB(21, 101, 192) ' Borders .Borders.LineStyle = xlContinuous .Borders.Weight = xlMedium .Borders.Color = RGB(0, 0, 0) ' Alignment .HorizontalAlignment = xlCenter .VerticalAlignment = xlVCenter .WrapText = True ' Merge cells .Merge End With

UsedRange & Dynamic Data Processing

' Find the actual data range dynamically Dim ws As Worksheet Set ws = ActiveSheet ' UsedRange = rectangle covering all used cells Dim used As Range Set used = ws.UsedRange MsgBox "Data: " & used.Rows.Count & " rows x " & used.Columns.Count & " cols" ' Find last row with data in column A Dim lastRow As Long lastRow = ws.Cells(ws.Rows.Count, 1).End(xlUp).Row ' Process all data rows (skip header row 1) Dim i As Long For i = 2 To lastRow With ws.Cells(i, 5) ' Column E .Value = ws.Cells(i, 3).Value * ws.Cells(i, 4).Value End With Next i ' Auto-fit columns after filling data ws.UsedRange.EntireColumn.AutoFit
✅ Speed TipWhen writing many cells, read the whole range into an array first, process in memory, then write back. Can be 100x faster than cell-by-cell access!

🎬 Demo — Reading & Writing a Data Table

Scenario: We have sales data in columns A–C. Write a macro to calculate total (qty × price) in column D for all rows.
Sub CalcTotals() Dim ws As Worksheet Dim lastRow As Long Dim i As Long Set ws = ActiveSheet lastRow = ws.Cells( _ ws.Rows.Count, 1).End(xlUp).Row ' Write header ws.Cells(1, 4).Value = "Total" ' Calculate each row For i = 2 To lastRow ws.Cells(i, 4).Value = _ ws.Cells(i, 2) * ws.Cells(i, 3) Next i ws.Columns("D").AutoFit End Sub

📋 Key Takeaways

  • ThisWorkbook = your file; ActiveWorkbook = current active file
  • Range("A1") and Cells(row, col) are two ways to reference cells
  • Offset() and Resize() navigate relative to a cell
  • Always use Cells(Rows.Count,1).End(xlUp).Row for dynamic last row
  • Turn off ScreenUpdating for faster macros
  • Application.WorksheetFunction gives access to Excel functions

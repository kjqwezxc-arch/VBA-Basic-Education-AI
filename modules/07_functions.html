Built-in Functions

String, Math, Date functions, MsgBox, InputBox, and WorksheetFunction

String Functions — Text Manipulation

FunctionWhat it doesExample → Result
Len(s)Length of stringLen("Hello") → 5
Left(s,n)First n charactersLeft("Yamaha",3) → "Yam"
Right(s,n)Last n charactersRight("Yamaha",3) → "aha"
Mid(s,start,n)n chars from position startMid("Yamaha",2,3) → "ama"
InStr(s,find)Find position (0 = not found)InStr("Yamaha","ah") → 4
Replace(s,old,new)Replace textReplace("Hi World","World","VBA") → "Hi VBA"
Trim(s)Remove leading/trailing spacesTrim(" Hello ") → "Hello"
UCase(s) / LCase(s)Upper/lower caseUCase("hello") → "HELLO"
Split(s,delim)Split into arraySplit("A,B,C",",") → ["A","B","C"]
Join(arr,delim)Join array to stringJoin(arr,"-") → "A-B-C"
Format(val,fmt)Format as stringFormat(1234,"#,##0") → "1,234"

String Functions — Code Examples

Sub StringExamples() Dim s As String s = " MT-09 Sport 2024 " ' Clean the string s = Trim(s) ' Remove spaces → "MT-09 Sport 2024" s = UCase(s) ' → "MT-09 SPORT 2024" ' Extract parts Dim model As String model = Left(s, 5) ' → "MT-09" ' Check if contains a word If InStr(s, "2024") > 0 Then MsgBox "This is a 2024 model" End If ' Split a CSV-like string Dim parts() As String parts = Split("Tanaka,Suzuki,Yamamoto", ",") ' parts(0)="Tanaka", parts(1)="Suzuki", parts(2)="Yamamoto" Dim i As Long For i = 0 To UBound(parts) Cells(i+1, 1).Value = parts(i) Next i End Sub

Math & Conversion Functions

' Math functions Abs(-42) ' → 42 (absolute value) Int(3.9) ' → 3 (integer part) Round(3.567, 2) ' → 3.57 Sqr(16) ' → 4 (square root) Fix(3.9) ' → 3 (truncate) ' Random numbers Rnd ' 0 to 1 Int((100 - 1 + 1) * Rnd + 1) ' → random 1 to 100 ' Type conversion CInt("42") ' String → Integer CLng("100000") ' String → Long CDbl("3.14") ' String → Double CStr(42) ' Number → String CBool(1) ' → True
' Date functions Date ' Today's date Now ' Current date + time Time ' Current time Year(Date) ' Current year Month(Date) ' Current month Day(Date) ' Day of month Weekday(Date) ' Day of week (1=Sun) DateAdd("d", 30, Date) ' 30 days from now DateDiff("d", #1/1/2024#, Date) ' Days since Jan 1 2024 Format(Date, "yyyy/mm/dd") ' → "2026/05/04" IsDate("2026/1/1") ' True IsNumeric("42") ' True

MsgBox — Show Messages & Dialogs

' Simple message MsgBox "Process complete!" ' With title and icon MsgBox "File saved!", vbInformation, "Success" MsgBox "Delete all data?", vbExclamation, "Warning" ' Yes/No dialog — check the response Dim answer As Integer answer = MsgBox("Are you sure?", vbYesNo + vbQuestion, "Confirm") If answer = vbYes Then ' User clicked Yes Else Exit Sub ' User clicked No End If ' Button constants you can combine with + ' vbOKOnly, vbOKCancel, vbYesNo, vbYesNoCancel ' vbInformation, vbQuestion, vbExclamation, vbCritical
📌 MsgBox Return ValuesvbOK=1, vbCancel=2, vbYes=6, vbNo=7, vbAbort=3, vbRetry=4, vbIgnore=5

InputBox — Get User Input

' Basic InputBox Dim userName As String userName = InputBox("Enter your name:", "Welcome") If userName = "" Then Exit Sub ' User cancelled MsgBox "Hello, " & userName ' InputBox with default value Dim rate As Double rate = CDbl(InputBox("Enter tax rate (%):", "Tax", "10")) ' Application.InputBox with type checking (more powerful) Dim targetRange As Range On Error Resume Next Set targetRange = Application.InputBox( _ "Select the data range:", _ "Select Range", _ Type:=8) ' Type 8 = Range selection! On Error GoTo 0 If targetRange Is Nothing Then Exit Sub MsgBox "You selected: " & targetRange.Address
📌 Application.InputBox Type Values0=Formula, 1=Number, 2=String, 4=Boolean, 8=Range, 16=Error value

Application.WorksheetFunction — Use Excel Functions in VBA

' Access Excel's built-in functions from VBA! Dim ws As Worksheet Set ws = ActiveSheet ' SUM, AVERAGE, MAX, MIN Dim total As Double total = Application.WorksheetFunction.Sum(Range("A1:A100")) Dim avg As Double avg = Application.WorksheetFunction.Average(Range("B1:B50")) ' VLOOKUP Dim result As Variant On Error Resume Next result = Application.WorksheetFunction.VLookup( _ "MT-09", Range("A:C"), 3, False) On Error GoTo 0 If IsEmpty(result) Then result = "Not Found" ' COUNTIF Dim cnt As Long cnt = Application.WorksheetFunction.CountIf(Range("B:B"), "Pass") ' Shorthand using WorksheetFunction Dim wf As WorksheetFunction Set wf = Application.WorksheetFunction total = wf.Sum(Range("A1:A10"))

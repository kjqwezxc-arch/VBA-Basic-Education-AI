Getting Started with VBE

Enable the Developer Tab, explore the Visual Basic Editor,
and write your very first macro

Step-by-Step Hands-On

Step 1: Enable the Developer Tab

The Developer tab is hidden by default. Here's how to show it:

1
Open Excel and click File in the top-left corner
2
Click Options at the bottom of the left panel
3
In the Excel Options window, select Customize Ribbon
4
On the right side under Main Tabs, check the box for Developer
5
Click OK — the Developer tab now appears in the ribbon!
✅ You'll know it worked when… You can see "Developer" between "View" and "Help" in the Excel ribbon tabs.

Step 2: Opening the Visual Basic Editor

There are 3 ways to open the VBE:

A
Keyboard shortcut: Press Alt + F11 — fastest method!
B
Developer Tab: Click Developer → Visual Basic button
C
Right-click sheet tab: Right-click any sheet tab → View Code
⌨️ Remember This! Alt + F11 is the fastest way. Use it constantly!

VBE Interface Tour

The Visual Basic Editor has 4 key areas:

🗂️ 1. Project Explorer (Ctrl+R)

Shows all the "objects" in your workbook — sheets, ThisWorkbook, and Modules. Think of it as the file tree for your VBA code.

📝 2. Code Window

Where you type your VBA code. Double-click any item in Project Explorer to open its code window.

📋 3. Properties Window (F4)

Shows properties of the selected object. Useful for UserForms and controls. Change names, captions, sizes here.

⌨️ 4. Immediate Window (Ctrl+G)

A "scratchpad" to test single lines of code and see output instantly. Essential for debugging. Type ? 2+2 and press Enter!

Step 3: Add a Module & Write Code

All standard macros go in a Module. Here's how to create one:

1
In VBE, click Insert menu → Module
2
A new "Module1" appears in Project Explorer under Modules
3
The Code Window opens — you can start typing your Sub here

📐 Basic Structure of a Macro

Sub MacroName() ' Your code goes between Sub and End Sub ' Lines starting with ' are comments (ignored when running) MsgBox "This is your first macro!" End Sub
📌 Naming Rules • No spaces in macro names (use underscores: My_Macro or CamelCase: MyMacro)  • Start with a letter  • Max 255 characters

Step 4: Running Your Macro

There are several ways to run a macro:

⌨️ From VBE

Place cursor inside the Sub → Press F5

📋 From Excel (Alt+F8)

In Excel, press Alt+F8 → Select macro → Click Run

🔘 Button on Sheet

Developer tab → Insert → Button (Form Controls) → Draw → Assign macro

⚡ Keyboard Shortcut

When recording or setting up a macro, assign a shortcut like Ctrl+Shift+M

📌 Quick Access Toolbar

Right-click the toolbar → Customize → Add macro as a button

⚠️ Save as .xlsm! Always save files with macros as Excel Macro-Enabled Workbook (.xlsm) — regular .xlsx files cannot store VBA code.

Bonus: Record a Macro (No Coding!)

Excel can record your actions and convert them to VBA automatically — great for learning!

1
Go to Developer tab → Click Record Macro
2
Give it a name and optional shortcut → Click OK
3
Perform actions in Excel (format cells, type data, apply filters…)
4
Click Stop Recording when done
5
Open VBE → Look at the generated code in the module!
💡 Learning Hack Record a macro doing something, then read the generated VBA. This is one of the fastest ways to learn VBA syntax for Excel-specific tasks!

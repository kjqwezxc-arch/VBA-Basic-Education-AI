💻 VBA Code — Initialize & Auto Part ID

' ═══════════════════════════════════ ' MODULE: frmPartsEntry (UserForm code) ' ═══════════════════════════════════ Private Sub UserForm_Initialize() ' ── Auto-generate next Part ID ── Dim ws As Worksheet Set ws = ThisWorkbook.Sheets("Database") Dim lastRow As Long lastRow = ws.Cells(ws.Rows.Count, 1).End(xlUp).Row Dim nextNum As Long nextNum = lastRow ' Row 1 = header txtPartID.Text = "YMH-" & Format(nextNum, "000") txtPartID.Enabled = False ' Read-only ' ── Populate ComboBoxes ── With cboCategory .AddItem "Engine" .AddItem "Brakes" .AddItem "Drive" .AddItem "Electrical" .AddItem "Body & Frame" .AddItem "Suspension" End With ' Load suppliers from Reference sheet Dim wsRef As Worksheet Set wsRef = ThisWorkbook.Sheets("Reference") Dim i As Long For i = 2 To wsRef.Cells(wsRef.Rows.Count, 1).End(xlUp).Row cboSupplier.AddItem wsRef.Cells(i, 1).Value Next i With cboStatus .AddItem "Active" .AddItem "Inactive" .AddItem "Discontinued" .ListIndex = 0 End With ' Update status bar with record count lblStatus.Caption = "Ready — " & (lastRow - 1) & " records in Database" End Sub
📌 Auto Part ID LogicCount rows in Database → subtract 1 (header) → format as 3-digit number → prefix "YMH-". Each new entry gets a unique, sequential ID automatically.
✅ Loading ComboBox from SheetInstead of hardcoding supplier names in VBA, load them from a "Reference" sheet. Adding a new supplier only requires editing the sheet — no VBA changes needed.
⚠️ Always set .ListIndex = 0Without this, the ComboBox shows blank on open. Set to 0 to pre-select the first item and avoid "no selection" validation errors.

💾 VBA Code — Validate & Save to Sheet

Private Sub btnSave_Click() ' ── STEP 1: Validate required fields ── If Trim(txtPartName.Text) = "" Then MsgBox "Part Name is required!", vbExclamation, "Validation Error" txtPartName.SetFocus : Exit Sub End If If cboCategory.Value = "" Then MsgBox "Please select a Category!", vbExclamation, "Validation Error" cboCategory.SetFocus : Exit Sub End If If Not IsNumeric(txtQty.Text) Or Trim(txtQty.Text) = "" Then MsgBox "Qty must be a valid number!", vbExclamation, "Validation Error" txtQty.SetFocus : Exit Sub End If If CDbl(txtQty.Text) < 0 Then MsgBox "Qty cannot be negative!", vbExclamation : Exit Sub End If ' ── STEP 2: Find next empty row ── Dim ws As Worksheet Set ws = ThisWorkbook.Sheets("Database") Dim nRow As Long nRow = ws.Cells(ws.Rows.Count, 1).End(xlUp).Row + 1 ' ── STEP 3: Write all fields ── Application.ScreenUpdating = False ws.Cells(nRow, 1).Value = txtPartID.Text ' A: Part ID ws.Cells(nRow, 2).Value = txtPartName.Text ' B: Part Name ws.Cells(nRow, 3).Value = cboCategory.Value ' C: Category ws.Cells(nRow, 4).Value = CLng(txtQty.Text) ' D: Qty ws.Cells(nRow, 5).Value = CDbl(txtPrice.Text) ' E: Price ws.Cells(nRow, 6).Value = cboSupplier.Value ' F: Supplier ws.Cells(nRow, 7).Value = IIf(txtReorder.Text <> "", CLng(txtReorder.Text), 0) ws.Cells(nRow, 8).Value = cboStatus.Value ' H: Status ws.Cells(nRow, 9).Value = Now() ' I: Timestamp ws.Cells(nRow, 10).Value = Environ("USERNAME") ' J: Windows user ws.Cells(nRow, 11).Value = txtRemarks.Text ' K: Remarks ws.Cells(nRow, 12).Value = "" ' L: Synced = blank (not yet) Application.ScreenUpdating = True ' ── STEP 4: Confirm & refresh ── MsgBox "✅ Part ID " & txtPartID.Text & " saved!", vbInformation, "Saved" Call ClearForm Call UserForm_Initialize ' Refresh ID and counters End Sub Private Sub ClearForm() txtPartName.Text = "" : txtQty.Text = "" txtPrice.Text = "" : txtRemarks.Text = "" txtReorder.Text = "" : cboCategory.ListIndex = -1 cboSupplier.ListIndex = -1 : cboStatus.ListIndex = 0 End Sub

🔍 Search & Update Scattered Records

The Search tab lets you find a record by Part ID or Name, load it into the form, edit it, and write updates back — including updating every matching row across the sheet.

' ── Search by Part ID or Name ── Private Sub btnSearch_Click() Dim ws As Worksheet Set ws = ThisWorkbook.Sheets("Database") Dim searchVal As String searchVal = Trim(txtSearch.Text) If searchVal = "" Then MsgBox "Enter a Part ID or Name to search.", vbInformation Exit Sub End If ' Use Excel's Find method (faster than loop) Dim foundCell As Range Set foundCell = ws.Columns(1).Find( _ What:=searchVal, LookIn:=xlValues, _ LookAt:=xlWhole, MatchCase:=False) ' Try column B (Part Name) if not in A If foundCell Is Nothing Then Set foundCell = ws.Columns(2).Find( _ What:=searchVal, LookIn:=xlValues, _ LookAt:=xlPart, MatchCase:=False) End If If foundCell Is Nothing Then MsgBox "No match found for: " & searchVal, vbExclamation Exit Sub End If ' ── Load record into the form fields ── Dim r As Long r = foundCell.Row txtPartID.Text = ws.Cells(r, 1).Value txtPartName.Text = ws.Cells(r, 2).Value cboCategory.Value = ws.Cells(r, 3).Value txtQty.Text = ws.Cells(r, 4).Value txtPrice.Text = ws.Cells(r, 5).Value cboSupplier.Value = ws.Cells(r, 6).Value txtReorder.Text = ws.Cells(r, 7).Value cboStatus.Value = ws.Cells(r, 8).Value txtRemarks.Text = ws.Cells(r, 11).Value ' Store row number for the Update button lblFoundRow.Tag = r lblStatus.Caption = "Found: Row " & r & " — edit & click Update" End Sub
' ── Update SCATTERED matching records ── ' Use case: supplier name changed company-wide, ' update ALL rows with the old supplier name Sub UpdateScatteredRecords( _ oldSupplier As String, _ newSupplier As String) Dim ws As Worksheet Set ws = ThisWorkbook.Sheets("Database") Dim lastRow As Long lastRow = ws.Cells(ws.Rows.Count,1).End(xlUp).Row Dim updCount As Long updCount = 0 Application.ScreenUpdating = False Dim i As Long For i = 2 To lastRow If UCase(ws.Cells(i,6).Value) = UCase(oldSupplier) Then ws.Cells(i,6).Value = newSupplier ws.Cells(i,9).Value = Now() ' Update timestamp ws.Cells(i,10).Value = Environ("USERNAME") ws.Cells(i,12).Value = "" ' Mark as unsynced updCount = updCount + 1 End If Next i Application.ScreenUpdating = True MsgBox updCount & " records updated from '" _ & oldSupplier & "' → '" & newSupplier & "'", _ vbInformation, "Batch Update Complete" End Sub ' ── How to call from the form ── Private Sub btnBatchUpdate_Click() Dim oldVal As String, newVal As String oldVal = InputBox("Enter OLD supplier name:") If oldVal = "" Then Exit Sub newVal = InputBox("Enter NEW supplier name:") If newVal = "" Then Exit Sub Call UpdateScatteredRecords(oldVal, newVal) End Sub

📤 Cross-Workbook Data Transfer — Copy to MasterDB

This is the Sync tab workflow — copy all unsynced rows (Column L = blank) from Inventory.xlsm into MasterDB.xlsx and mark them as synced.

' ═══════════════════════════════════════════════════════════ ' SyncToMasterDB — copy new rows to the central database file ' ═══════════════════════════════════════════════════════════ Sub SyncToMasterDB() Dim masterPath As String masterPath = ThisWorkbook.Path & "\MasterDB.xlsx" ' ── Check the file exists ── If Dir(masterPath) = "" Then MsgBox "MasterDB.xlsx not found at: " & masterPath, vbCritical Exit Sub End If ' ── Open MasterDB (hidden) ── Application.ScreenUpdating = False Application.DisplayAlerts = False Dim wbMaster As Workbook Set wbMaster = Workbooks.Open(masterPath, ReadOnly:=False) Dim wsSrc As Worksheet ' Source: local Database sheet Dim wsDst As Worksheet ' Destination: MasterDB sheet Set wsSrc = ThisWorkbook.Sheets("Database") Set wsDst = wbMaster.Sheets("MasterDB") Dim srcLast As Long srcLast = wsSrc.Cells(wsSrc.Rows.Count, 1).End(xlUp).Row Dim dstNext As Long dstNext = wsDst.Cells(wsDst.Rows.Count, 1).End(xlUp).Row + 1 Dim copyCount As Long copyCount = 0 Dim i As Long ' ── Loop: copy only rows where Col L = "" (not yet synced) ── For i = 2 To srcLast If Trim(wsSrc.Cells(i, 12).Value) = "" Then ' Copy entire row (columns A:K) using array for speed Dim rowData As Variant rowData = wsSrc.Range(wsSrc.Cells(i, 1), wsSrc.Cells(i, 11)).Value wsDst.Range(wsDst.Cells(dstNext, 1), wsDst.Cells(dstNext, 11)).Value = rowData ' Mark source row as synced with date wsSrc.Cells(i, 12).Value = "Synced " & Format(Now(), "yyyy-mm-dd hh:mm") dstNext = dstNext + 1 copyCount = copyCount + 1 End If Next i ' ── Save and close MasterDB ── wbMaster.Save wbMaster.Close Application.ScreenUpdating = True Application.DisplayAlerts = True MsgBox copyCount & " new record(s) synced to MasterDB.xlsx!", _ vbInformation, "Sync Complete" lblStatus.Caption = "Last sync: " & Format(Now(), "yyyy-mm-dd hh:mm") End Sub

📥 Import Data FROM Another Workbook

Let users pick any Excel file and pull selected data into the local Database — handles files that may not be open.

' ── Let user browse for a file ── Sub ImportFromFile() Dim importPath As String importPath = Application.GetOpenFilename( _ FileFilter:="Excel Files (*.xlsx;*.xlsm), *.xlsx;*.xlsm", _ Title:="Select Source Workbook") If importPath = "False" Then Exit Sub ' User cancelled Application.ScreenUpdating = False Dim wbSrc As Workbook Set wbSrc = Workbooks.Open(importPath, ReadOnly:=True) Dim wsImport As Worksheet Set wsImport = wbSrc.Sheets(1) ' First sheet Dim wsDst As Worksheet Set wsDst = ThisWorkbook.Sheets("Database") Dim srcLast As Long srcLast = wsImport.Cells(wsImport.Rows.Count, 1).End(xlUp).Row Dim dstNext As Long dstNext = wsDst.Cells(wsDst.Rows.Count, 1).End(xlUp).Row + 1 Dim importCount As Long : importCount = 0 Dim i As Long ' ── Copy rows, skip if Part ID already exists ── For i = 2 To srcLast Dim partID As String partID = Trim(wsImport.Cells(i, 1).Value) If partID = "" Then GoTo NextRow ' Check for duplicate in destination Dim dupCheck As Range Set dupCheck = wsDst.Columns(1).Find(partID, LookAt:=xlWhole) If Not dupCheck Is Nothing Then GoTo NextRow ' Skip duplicates ' Copy the row data wsDst.Cells(dstNext, 1).Resize(1, 11).Value = _ wsImport.Cells(i, 1).Resize(1, 11).Value wsDst.Cells(dstNext, 12).Value = "" ' Mark unsynced dstNext = dstNext + 1 importCount = importCount + 1 NextRow: Next i wbSrc.Close SaveChanges:=False Application.ScreenUpdating = True MsgBox importCount & " new records imported (duplicates skipped).", _ vbInformation, "Import Complete" End Sub

Key Techniques Used

GetOpenFilename
Shows the standard Windows file browser. Returns the selected path as a string, or the string "False" if cancelled. Always check for cancellation!
ReadOnly:=True
Opens the source file without locking it for others. Always open source files in ReadOnly mode — close them with SaveChanges:=False.
Duplicate Detection
Before copying, use Find on Column A to check if the Part ID already exists. This prevents importing the same data twice.
⚠️ Always close opened workbooksUse wbSrc.Close SaveChanges:=False even if an error occurs. Add error handling with On Error GoTo CleanUp and put the Close in the CleanUp label.

📊 Auto-Generate Summary Report

One click rebuilds the Summary sheet — total stock value, items by category, and low-stock alerts.

' ═══════════════════════════════════════════════════════ ' GenerateSummary — clears and rebuilds the Summary sheet ' ═══════════════════════════════════════════════════════ Sub GenerateSummary() Dim wsDB As Worksheet : Set wsDB = ThisWorkbook.Sheets("Database") Dim wsSum As Worksheet : Set wsSum = ThisWorkbook.Sheets("Summary") Application.ScreenUpdating = False ' ── Clear old summary ── wsSum.Cells.Clear ' ── Write headers ── wsSum.Range("A1").Value = "Parts Inventory Summary" wsSum.Range("A1").Font.Bold = True wsSum.Range("A1").Font.Size = 14 wsSum.Range("A2").Value = "Generated: " & Format(Now(), "yyyy-mm-dd hh:mm") wsSum.Range("A2").Font.Color = RGB(100, 100, 100) ' ── KPI block ── Dim lastRow As Long lastRow = wsDB.Cells(wsDB.Rows.Count, 1).End(xlUp).Row Dim totalItems As Long totalItems = lastRow - 1 ' Exclude header row Dim totalValue As Double totalValue = Application.WorksheetFunction.SumProduct( _ wsDB.Range("D2:D" & lastRow), wsDB.Range("E2:E" & lastRow)) wsSum.Range("A4").Value = "Total SKUs" wsSum.Range("B4").Value = totalItems wsSum.Range("A5").Value = "Total Stock Value (¥)" wsSum.Range("B5").Value = totalValue wsSum.Range("B5").NumberFormat = "¥#,##0" ' ── Category breakdown using Dictionary ── Dim dict As Object Set dict = CreateObject("Scripting.Dictionary") Dim i As Long For i = 2 To lastRow Dim cat As String cat = wsDB.Cells(i, 3).Value If cat <> "" Then dict(cat) = dict(cat) + wsDB.Cells(i, 4).Value ' Sum qty End If Next i wsSum.Range("A7").Value = "Category" : wsSum.Range("A7").Font.Bold = True wsSum.Range("B7").Value = "Total Qty" : wsSum.Range("B7").Font.Bold = True Dim r As Long : r = 8 Dim key As Variant For Each key In dict.Keys wsSum.Cells(r, 1).Value = key wsSum.Cells(r, 2).Value = dict(key) r = r + 1 Next key ' ── Low stock alerts (Qty < Reorder Point) ── wsSum.Cells(r + 1, 1).Value = "⚠️ LOW STOCK ALERTS" wsSum.Cells(r + 1, 1).Font.Bold = True wsSum.Cells(r + 1, 1).Font.Color = RGB(198, 40, 40) Dim alertRow As Long : alertRow = r + 2 For i = 2 To lastRow If wsDB.Cells(i,4).Value < wsDB.Cells(i,7).Value Then ' Qty < ReorderPt wsSum.Cells(alertRow, 1).Value = wsDB.Cells(i,1).Value ' Part ID wsSum.Cells(alertRow, 2).Value = wsDB.Cells(i,2).Value ' Name wsSum.Cells(alertRow, 3).Value = wsDB.Cells(i,4).Value ' Current Qty wsSum.Cells(alertRow, 4).Value = wsDB.Cells(i,7).Value ' Reorder Pt wsSum.Cells(alertRow, 1).Font.Color = RGB(198, 40, 40) alertRow = alertRow + 1 End If Next i Application.ScreenUpdating = True MsgBox "Summary sheet updated!", vbInformation End Sub

📦 Archive Workflow — Move Old Records

Move discontinued/inactive parts from Database → Archive sheet, keeping Database clean and fast.

' ── Archive all "Discontinued" parts ── Sub ArchiveDiscontinued() Dim wsDB As Worksheet Dim wsArc As Worksheet Set wsDB = ThisWorkbook.Sheets("Database") Set wsArc = ThisWorkbook.Sheets("Archive") Dim confirm As Integer confirm = MsgBox("Archive ALL Discontinued parts?" & _ vbCrLf & "This will MOVE them to the Archive sheet.", _ vbYesNo + vbQuestion, "Confirm Archive") If confirm <> vbYes Then Exit Sub Application.ScreenUpdating = False Dim lastRow As Long lastRow = wsDB.Cells(wsDB.Rows.Count, 1).End(xlUp).Row Dim arcNext As Long arcNext = wsArc.Cells(wsArc.Rows.Count, 1).End(xlUp).Row + 1 ' Loop BACKWARDS — safe when deleting rows! Dim i As Long Dim movedCount As Long : movedCount = 0 For i = lastRow To 2 Step -1 If wsDB.Cells(i, 8).Value = "Discontinued" Then ' Copy row to Archive wsDB.Rows(i).Copy wsArc.Rows(arcNext) ' Add archive timestamp in column M wsArc.Cells(arcNext, 13).Value = "Archived " & Format(Now(), "yyyy-mm-dd") ' Delete original row from Database wsDB.Rows(i).Delete arcNext = arcNext + 1 movedCount = movedCount + 1 End If Next i Application.ScreenUpdating = True MsgBox movedCount & " part(s) archived successfully.", _ vbInformation, "Archive Complete" End Sub

⚠️ Critical: Loop Backwards when Deleting

❌ WRONG — loops forward and skips rows:

For i = 2 To lastRow
  If ... Then wsDB.Rows(i).Delete
Next i

When row 5 is deleted, row 6 becomes row 5 — the loop skips it!

✅ CORRECT — loops backward:

For i = lastRow To 2 Step -1
  If ... Then wsDB.Rows(i).Delete
Next i

Deleting from the bottom up never affects the rows above that you haven't processed yet.

✅ Restore from ArchiveBuild a corresponding RestoreFromArchive macro that reverses this — copy from Archive back to Database and delete from Archive. Same backward-loop pattern applies.

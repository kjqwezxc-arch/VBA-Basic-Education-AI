Control Flow

Make decisions and repeat actions — If/Then, Loops, and Select Case

If / Then / Else

Make decisions in your code — run different code based on a condition:

' Basic If/Then If score >= 60 Then MsgBox "Pass!" End If ' If/Then/Else If score >= 60 Then result = "Pass" Else result = "Fail" End If ' If/ElseIf/Else If score >= 90 Then grade = "A" ElseIf score >= 80 Then grade = "B" ElseIf score >= 70 Then grade = "C" Else grade = "F" End If

Select Case — Cleaner Multiple Choices

When checking one variable against many values, Select Case is cleaner than many ElseIf statements:

Dim dept As String dept = "Engineering" Select Case dept Case "Sales" budget = 500000 Case "Engineering" budget = 1200000 Case "HR", "Admin" budget = 300000 Case Else budget = 200000 End Select
' Works with numbers too: Select Case score Case 90 To 100 grade = "A" Case 80 To 89 grade = "B" Case Is >= 70 grade = "C" Case Else grade = "F" End Select
💡 When to use Select CaseWhen you're comparing ONE variable against 3+ possible values. Cleaner than a chain of ElseIf.

For / Next Loop — Count from A to B

' Basic For/Next Dim i As Long For i = 1 To 5 Cells(i, 1).Value = i * 10 Next i ' Count backwards with Step For i = 10 To 1 Step -1 Debug.Print i Next i ' Count by 2s For i = 0 To 10 Step 2 ' i = 0, 2, 4, 6, 8, 10 Next i ' Exit loop early For i = 1 To 100 If Cells(i,1) = "" Then Exit For Next i

For Each — Loop Over Collections

For Each is used when you want to process every item in a collection (sheets, rows, cells, etc.):

' Loop over every worksheet Dim ws As Worksheet For Each ws In ThisWorkbook.Worksheets ws.Tab.Color = RGB(255, 143, 0) ' Set tab color Next ws ' Loop over a range of cells Dim cell As Range For Each cell In Range("A1:A10") If cell.Value < 0 Then cell.Interior.Color = RGB(255, 0, 0) ' Highlight negatives End If Next cell ' Loop over an array Dim names() As String names = Split("Tanaka,Suzuki,Yamamoto", ",") Dim n As Variant For Each n In names Debug.Print n Next n
✅ For Each vs For iUse For Each when you want every item in a collection. Use For i when you need the index number or want to skip items.

Do Loops — Run While a Condition is True

' Do While — check BEFORE running Dim row As Long row = 1 Do While Cells(row, 1) <> "" ' process the row row = row + 1 Loop ' Do Until — stop when TRUE row = 1 Do Until Cells(row, 1) = "" row = row + 1 Loop ' Do...Loop While — check AFTER running Dim answer As String Do answer = InputBox("Enter 'yes' to continue") Loop While answer <> "yes"
⚠️ Infinite Loop Warning!Always make sure your loop condition eventually becomes False, or use Exit Do to break out safely.
🔍 Finding the Last Row (common pattern)
Dim lastRow As Long lastRow = Cells(Rows.Count, 1) _ .End(xlUp).Row
This finds the last used row in column A — extremely useful!

The With Statement — Cleaner Code

With lets you perform multiple operations on one object without repeating its name:

❌ Without With (repetitive):

Range("A1").Font.Bold = True Range("A1").Font.Size = 14 Range("A1").Font.Color = RGB(255,0,0) Range("A1").Interior.Color = RGB(255,255,0) Range("A1").HorizontalAlignment = xlCenter

✅ With With (clean):

With Range("A1") .Font.Bold = True .Font.Size = 14 .Font.Color = RGB(255,0,0) .Interior.Color = RGB(255,255,0) .HorizontalAlignment = xlCenter End With
💡 Also nest With statementsYou can nest With inside another With for deeply nested objects like With ws.Cells(1,1).Font.

Nested Loops — Loops Inside Loops

Use nested loops to process 2D data (like rows AND columns):

Sub FillTable() Dim r As Long, c As Long ' Loop through rows 1-5 and columns 1-4 For r = 1 To 5 For c = 1 To 4 Cells(r, c).Value = r & "-" & c Next c Next r ' Result in spreadsheet: ' A1=1-1 B1=1-2 C1=1-3 D1=1-4 ' A2=2-1 B2=2-2 C2=2-3 D2=2-4 ' ...etc End Sub
📌 Performance TipWhen looping over thousands of cells, add Application.ScreenUpdating = False at the start and True at the end. This prevents screen flicker and runs much faster!

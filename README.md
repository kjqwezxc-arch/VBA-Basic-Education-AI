# VBA Education Module

This project is a self-contained educational module for learning VBA (Visual Basic for Applications), with a focus on Excel automation and practical programming skills. It includes interactive HTML modules, a Python-based launcher and server, and supporting scripts for easy use.

## Project Structure

- `index.html` — Main entry point for the course or module navigation.
- `launcher.py` — Python script to launch the course or supporting server.
- `server.py` — Backend server (if required for serving content or handling requests).
- `start.bat` — Batch file for easy startup on Windows.
- `modules/` — Contains all HTML modules for each lesson:
    - `01_intro.html` — Introduction to VBA
    - `02_getting_started.html` — Getting Started
    - `03_variables.html` — Variables
    - `04_control_flow.html` — Control Flow
    - `05_procedures.html` — Procedures
    - `06_excel_objects.html` — Excel Objects
    - `07_functions.html` — Functions
    - `08_error_handling.html` — Error Handling
    - `09_userforms.html` — UserForms
    - `10_practical.html` — Practical Exercises
    - `11_ai_intro.html` — AI Introduction
    - `12_ai_advanced.html` — Advanced AI Topics
    - `13_userform_automation.html` — UserForm Automation

## Getting Started

1. **Install Python 3.x** (if not already installed).
2. Run `start.bat` or execute `launcher.py` to start the course.
3. Open `index.html` in your browser to access the modules.

## Requirements
- Python 3.x
- (Optional) Any additional Python packages required by `server.py` or `launcher.py` (see code comments for details)
